# GCP Professional Cloud Architect — Study Hub

A self-contained study site for the Google Cloud **Professional Cloud Architect** exam (guide **v6.1**, updated Oct 30, 2025).

## Contents

- **`index.html`** — multi-page study hub: domain notes (Compute, Storage & Databases, Networking, Security & IAM, Operations & Cost, GenAI & ML, Well-Architected & Migration), the exam blueprint, a dedicated page per current case study (EHR Healthcare, Altostrat Media, Cymbal Retail, KnightMotives Automotive) with reveal-answer prep questions, and a 102-question self-grading practice quiz.
- **`cheatsheet.html`** — one-page keyword→service quick reference for last-minute review.

## Use it

Open `index.html` in any browser. Everything is inline — no build step, no dependencies.

To serve via **GitHub Pages**: repo Settings → Pages → deploy from `main` / root. The site will be available at `https://skalvinnathan.github.io/GCP-PCA/`.

---
*Service mappings are best-practice study guidance, not Google's official answer keys. Verify against [cloud.google.com/certification](https://cloud.google.com/learn/certification/cloud-architect).*
